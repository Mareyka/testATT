Проект представляет собой адаптивный лендинг для экскурсии "Тысячелетний Брест и Брестская крепость". 
Для запуска проекта склонируйте репозиторий на локальный компьютер: git clone, затем откройте папку 
проекта и запустите файл index.html в любом современном браузере - достаточно просто дважды 
кликнуть по файлу или использовать расширение Live Server в VS Code для удобной разработки с автоматическим обновлением. 
Все зависимости подгружаются автоматически, поэтому дополнительная установка пакетов не требуется. 
Проект полностью готов к просмотру и не требует сборки или установки зависимостей. Для проверки адаптивности 
используйте инструменты разработчика в браузере и переключайте режимы просмотра на desktop (от 1024px), 
tablet (от 768px) и mobile (от 360px).
